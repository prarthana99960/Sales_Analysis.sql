# 🛒 E-Commerce Data Analysis using MySQL

## Project Overview
This project demonstrates an end-to-end E-Commerce Sales Data Analysis using MySQL.  
The project includes:

- Database creation
- Table creation
- Data insertion
- Data cleaning
- SQL analysis queries
- Business insights generation

This project is useful for beginners and intermediate-level Data Analysts who want hands-on SQL practice.
